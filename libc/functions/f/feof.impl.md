###Synopsis

`#include <stdio.h>`

`int feof(FILE *stream);`

###Description

Tests the end-of-file indicator for the given stream.

Arguments:
<u>stream</u> - the stream to be tested.

###Return value

`1`if the end-of-file indicator is set  for the given stream,  `0` otherwise.

###Errors

No errors are defined.
